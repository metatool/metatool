# Metatool.UIElementsDetector.Tests

Unit and performance tests for the `Metatool.UIElementsDetector` library.

## Running Tests

Run the following command from the root directory or the test project directory:

```bash
dotnet test src/lib/Metatool.UIElementsDetector.Tests/Metatool.UIElementsDetector.Tests.csproj
```

## Requirements

*   A Windows environment with a display (for `ScreenCapture.NET`).
*   `icon_detect.onnx` model file (automatically copied from the library project).
